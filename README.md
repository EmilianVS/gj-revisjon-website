# Gyllstrøm & Johansen AS Website

Professional website for Gyllstrøm & Johansen AS - Statsautoriserte Revisorer i Drammen.

## Tech Stack
- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Lucide React

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deployment
The site is deployed to Vercel.
